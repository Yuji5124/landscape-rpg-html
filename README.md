# ランダム仲間RPG（プロトタイプ）

PCブラウザでプレイできる、ドラクエ風のコマンド選択式RPGのプロトタイプです。
HTML / CSS / JavaScript のみで動作し、外部ライブラリは使用していません。

## コンセプト

> ランダムに生まれた4人の仲間で20ステージを攻略し、
> 戦闘後のガチャで少しずつ強い仲間に入れ替えていくRPG。

毎回の戦闘後、「新しい仲間を入れるか／今の仲間を残すか」を選ぶ —
その選択を楽しむゲームです。

## 遊び方

1. `index.html` をブラウザ（Chrome / Edge / Firefox など）で開く。
2. 「はじめる」を押すと、レア度1中心の4人パーティがランダム生成される。
3. ステージ選択画面で遊べるステージを選ぶ。
4. ターン制コマンドバトル（攻撃 / スキル / 防御）で敵1体を倒す。
5. 勝利すると新しい仲間が1人ガチャ生成される。
   - パーティが4人未満ならそのまま加入。
   - 4人なら、誰かと入れ替えるか／さよならするかを選ぶ。
6. クリアすると次のステージが解放される。クリア済みは何度でも遊べる。

進行状況（クリア済みステージ・現在の4人パーティ）は `localStorage` に自動保存されます。

## 動作環境

- PCブラウザ（モダンブラウザ推奨）
- インストール不要。ファイルを開くだけで動作します。

## 実装メモ

- キャラクター名は起動時に100通り生成。
- ステータスはレア度とレベルから計算（`makeStats`）。
- スキルはレア度で解放され、`SKILL_POOL` から抽選される。
- 敵・ステージデータは `STAGES`（20件）で定義。
- 将来の経験値システムを見据え、`level` フィールドを保持。
