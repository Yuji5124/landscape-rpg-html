# assets

画像・音声などの素材を置くフォルダです。
プロトタイプ現在は絵文字と図形で代替しているため、素材ファイルはまだありません。

## 追加するときの想定構成

```
assets/
  characters/   キャラクター画像（例: c001.png）
  enemies/      敵画像（例: stage01.png）
  ui/           UIパーツ・アイコン
  bgm/          BGM
  se/           効果音
```

## 差し替えの目安

- 敵画像：`app.js` の `ENEMY_FIGURES`（絵文字）を `<img>` に置き換える。
- キャラ画像：キャラクターに `image` フィールドを追加し、各カードで参照する。

後から画像を足しやすいよう、表示部分はデータ参照で組んであります。
