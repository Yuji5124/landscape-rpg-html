# ゲームシステム詳細

## ゲーム形式

| 項目 | 内容 |
| --- | --- |
| ジャンル | RPG（ブラウザ） |
| 対応 | PCブラウザ |
| 戦闘方式 | ターン制コマンドバトル |
| パーティ人数 | 4人（常に最大4人） |
| 敵 | 各戦闘1体 |
| ステージ数 | 20 |

## キャラクター

各キャラクターは以下の情報を持つ（`app.js` の `createCharacter`）。

- 名前 / レア度 / レベル
- HP・MP（最大値と現在値）
- 攻撃力・防御力・素早さ
- スキル（最大4つ。スキル名で保持）

合計100通りの名前プールから生成される。

### レア度（1〜10）

| レア度 | 位置づけ |
| --- | --- |
| 1 | 弱いキャラクター |
| 2〜3 | 序盤向け |
| 4〜6 | 中盤向け |
| 7〜9 | 強力 |
| 10 | 非常に強力なレアキャラ |

レア度が高いほどステータスが高く、強いスキルを持つ。
ステータスは `makeStats(rarity, level)` でレア度・レベルから計算する。

### レベル（1〜10）

レベルは生成時の強さの目安で、**戦闘中に上がることはない（レベルアップ無し）**。

- 強くなる手段は「ガチャでより高レア／高レベルの仲間に入れ替える」こと。
  レベル上げ（グラインド）を不要にしたバランス。
- 後半ステージほど高レア・高レベルの仲間が手に入るため、自然とパーティが強くなる。
- ステータスはレア度とレベルから `makeStats(rarity, level)` で決まる。

### 体力（HP）

テンポを重視し、HPは控えめに設定（`makeStats` の `maxHp = 20 + rarity*8 + level*3`）。
1戦は概ね3〜5ターンで決着する。敵HP・攻撃力も合わせて控えめに調整している。

## スキル

`SKILL_POOL` で定義。各スキルは以下を持つ。

- スキル名 / 効果タイプ / 威力 / 消費MP / 対象 / 説明文 / 最低レア度

| スキル | タイプ | 説明 |
| --- | --- | --- |
| 通常攻撃 | attack | 基本の物理攻撃（全員所持） |
| 強攻撃 | strong | 威力の高い物理 |
| 回復 | heal | 味方1人を回復 |
| 防御アップ | buffDef | 自分の防御を上げる |
| 攻撃アップ | buffAtk | 自分の攻撃を上げる |
| 火属性攻撃 | fire | 防御を一部無視 |
| 雷属性攻撃 | thunder | 防御を一部無視・高威力 |
| 全体回復 | healAll | 味方全員を回復 |
| そせい | revive | 倒れた味方1人をHP60%で復活（レア度6〜） |
| 必殺技 | ultimate | 最大威力 |
| 復活の祈り | reviveAll | 倒れた味方全員をHP80%で復活（レア度9〜） |

### 生き返る手段（復活）

倒れた味方は次の手段で戦闘中に復活できる。

- **どうぐ「いのちの石」**：倒れた味方1人をHP半分で復活。
- **スキル「そせい」**（レア度6以上）：倒れた味方1人をHP60%で復活。
- **スキル「復活の祈り」**（レア度9以上）：倒れた味方全員をHP80%で復活。

復活系スキルは、倒れた味方がいないときは選べない（グレーアウト）。
なお戦闘と戦闘の間（ステージ選択時）は全員がHP/MP全快で復帰する。

スキルは最低レア度で解放され、レア度が高いほど強いスキルを持ちやすい。
通常攻撃は必ず所持し、残りはレア度に応じて抽選で最大4つまで（生成時に確定）。

## 属性・弱点システム

攻撃スキルには属性がある（火 🔥 / 雷 ⚡ / 物理）。

- 各ステージの敵には弱点属性が設定されている（無い敵もいる）。
- 弱点属性の攻撃を当てると **ダメージ1.6倍**＋「弱点をついた！」と表示。
- 弱点は戦闘開始時とステージ選択画面（敵名の横アイコン）で確認できる。
- 火属性攻撃・雷属性攻撃を持つ仲間を弱点に合わせて編成するのが攻略の鍵。

## 状態異常システム

毒（🟣）とまひ（⚡）がある。

- **毒**：ターン開始時に最大HPの約7%ダメージ（3ターン）。敵にも味方にも有効。
- **まひ**：50%の確率で行動できない（2ターン）。
- 味方スキル「毒攻撃」「しびれ攻撃」で敵に付与できる。
- ボスの特殊攻撃で味方が状態異常になることがある。
- どうぐ「どくけし草」で味方の状態異常を治せる。

## ボス戦（5ステージごと）

ステージ 5・10・15・20 はボス戦。

- 通常の敵よりHP約1.45倍・攻撃約1.1倍。図体も大きく【BOSS】表示。
- **1回の決戦で撃破すればクリア**（通常ステージの2〜3回撃破とは別扱い）。
- ボスは多彩な特殊行動を取る：通常攻撃 / 強烈な一撃 / 全体攻撃 /
  自己回復（HPが減ると使いやすい）/ 毒の一撃 / しびれの一撃。
- HPが30%以下になると「怒り」状態になり攻撃力が上がる。
- 弱点属性・状態異常・どうぐ・復活をフル活用するのが攻略のポイント。

## どうぐ（アイテム）

戦闘中に「どうぐ」コマンドで使える消費アイテム（`ITEM_POOL`）。使用するとその味方の1ターンを消費する。

| どうぐ | 効果 |
| --- | --- |
| やくそう | 味方1人のHPを40回復 |
| 上やくそう | 味方1人のHPを90回復 |
| まんげつ草 | 味方全員のHPを35回復 |
| まほうの水 | 味方1人のMPを30回復 |
| どくけし草 | 味方1人の状態異常（毒・まひ）を治す |
| いのちの石 | 倒れた味方1人をHP半分で復活 |
| ちからの実 | 味方1人の攻撃力を上げる |
| ばくだん石 | 敵に55の固定ダメージ |

- 開始時に「やくそう×4／まんげつ草／まほうの水／ばくだん石」を所持。
- 勝利するたびに1〜3個入手（後半ステージほど良いどうぐが出やすい）。
- 所持数はステージ選択画面の「どうぐ」バーに表示。`localStorage` に保存される。
- レベル上げが無いぶん、どうぐの使いどころが攻略の鍵になる。

## 戦闘

### 画面構成

- 上部：ステージ情報・敵・敵HP
- 中央：演出エリア（絵文字エフェクト）
- 下部左：メッセージウィンドウ
- 下部右：コマンド選択
- 右側：味方4人のステータス（名前・レア度・レベル・HP・MP）

### 流れ

1. ステージ選択
2. 敵1体が出現
3. 味方がパーティ順に行動（攻撃 / スキル / どうぐ / 防御）
4. 全員行動後、敵が攻撃（一定確率で強攻撃）
5. 敵HPが0で勝利 / 味方全滅で敗北
6. 勝利後にガチャで新しい仲間が1人生成
7. 仲間にする / 入れ替える / さよなら を選択
8. ステージクリア（次が解放）

ダメージ・回復は対象の上に数字がポップアップ表示される。

### ガチャの入れ替え判断

ガチャ画面では、新キャラと現在の4人それぞれの「総合力」スコアを並べて比較できる。

- スコアは `powerScore`（HP・攻・防・速・MP・レア度・スキル強度の加重和）。
- 各メンバーに、新キャラとの差を ▲（強くなる）／▼（弱くなる）で色分け表示。
- 入れ替え時は、強化になるメンバーのカードを緑枠で強調する。
- 仲間の名前はパーティ内で重複しないように生成される。

### ダメージ計算（概要）

- 物理：`攻撃力 × 威力 − 敵防御` に乱数（±10%）
- 属性（火/雷/必殺）：敵防御を40%無視
- 回復：威力 + レア度・レベル補正
- 防御コマンド：そのラウンドの被ダメージを半減

## ステージ

`STAGES`（20件）で定義。後半ほど敵のHP・攻撃・防御・素早さが上がる。
一度クリアしたステージは何度でも遊べる。

### 撃破回数（クリア条件）

各ステージは規定回数を撃破しないと「クリア扱い（次ステージ解放）」にならない。

- 必要撃破回数 `winsRequired`：ステージ1〜10は **2回**、11〜20は **3回**。
- 撃破ごとに経験値とガチャがもらえるので、繰り返すたびにパーティが育つ。
- 進捗は `stageProgress`（ステージ番号→撃破回数）で保存。
  ステージ選択画面では `●○` のピップで進捗を表示する。

### ステージ別 仲間ガチャのレア度範囲

| ステージ | レア度範囲 |
| --- | --- |
| 1〜2 | 1 |
| 3〜4 | 1〜2 |
| 5〜6 | 1〜3 |
| 7〜8 | 1〜4 |
| 9〜10 | 2〜5 |
| 11〜12 | 2〜6 |
| 13〜14 | 3〜7 |
| 15〜16 | 3〜8 |
| 17〜18 | 4〜9 |
| 19〜20 | 5〜10 |

高レアほど低確率（`weightedRarity` で指数的に重み付け）。

## セーブ

`localStorage`（キー: `random_party_rpg_save_v1`）に保存。

- クリア済みステージ
- 現在の4人パーティ
- ユニークID採番カウンタ

読み込み時はHP/MPを全回復した状態で復帰する。
